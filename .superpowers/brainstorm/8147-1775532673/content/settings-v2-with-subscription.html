变化点

  • Tab 切换 API Key / Subscription：两栏并行展示
  • Subscription 自动检测：Claude 从 ~/.claude/credentials.json 读，Gemini 走 Google OAuth 一键登录
  • "Auto" / "OAuth" 标签：清楚区分认证方式
  • 每个 agent 显示 "via 哪个 provider"：能看到这个 agent 现在在用 subscription 还是 API key，以及大致成本
  • Routing Strategy：默认 "prefer subscription"（省钱），quota 用完自动 fallback API key
  • 实施工作量：相比之前 +4 个 subscription adapter，约 +8-12 task（每个 subscription provider 一个 adapter + auth flow + UI）
